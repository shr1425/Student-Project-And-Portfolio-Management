# 🎓 Student Project & Portfolio Management System

## 📌 Overview
The **Student Project & Portfolio Management System** is a full-stack web application designed to streamline the management of student projects and portfolios within an academic environment. It enables students to showcase their work, while faculty and administrators can efficiently monitor, evaluate, and manage project progress.

---

## 🚀 Key Features

### 👨‍🎓 Student Module
- Register and login securely
- Create and manage personal portfolio
- Submit and track academic projects
- View project status and feedback

### 👨‍🏫 Faculty Module
- Manage student project submissions
- Review and evaluate projects
- Provide feedback and updates

### 🛠 Admin Module
- Manage students and faculty records
- Monitor system activities
- Approve/reject project submissions
- Maintain overall system control
